X - Koordinate / Abstand

- Eingabe eines Zahlenwertes mit der Tastatur: zB "12.5"
- Eingabe einer Variablen (Tastatur oder Objektbrowser); zB "V20"
- Eingabe von Punkt - ergibt X-Koordinate dieses Punktes zB "X(P20)"
- Eingabe eines Vektors - ergibt X-Koordinate dieses Vektors zB "X(D20)"
- Eingabe einer Line ergibt die X-Koordinatendifferenz dieser Linie zB "X(L20)"
  
  

____________________________________________________________________________



Y - Koordinate / Abstand

- Eingabe eines Zahlenwertes mit der Tastatur: zB "12.5"
- Eingabe einer Variablen (Tastatur oder Objektbrowser); zB "V20"
- Eingabe von Punkt - ergibt Y-Koordinate dieses Punktes zB "Y(P20)"
- Eingabe eines Vektors - ergibt Y-Koordinate dieses Vektors zB "Y(D20)"
- Eingabe einer Line ergibt die Y-Koordinatendifferenz dieser Linie zB "Y(L20)"
  


____________________________________________________________________________



Z - Koordinatei / Abstand

- Eingabe eines Zahlenwertes mit der Tastatur: zB "12.5"
- Eingabe einer Variablen (Tastatur oder Objektbrowser); zB "V20"
- Eingabe von Punkt - ergibt Z-Koordinate dieses Punktes zB "Z(P20)"
- Eingabe eines Vektors - ergibt Z-Koordinate dieses Vektors zB "Z(D20)"
- Eingabe einer Line ergibt die Z-Koordinatendifferenz dieser Linie zB "Z(L20)"
  


____________________________________________________________________________



Zahlenwert (Value)

- Eingabe eines Zahlenwertes mit der Tastatur:
  - direkt: zB "12.5"
  - aus Variable: zB "V20"
  - Länge eines Vektor: zB "D23"
- Selektion einer Linie ergibt die Länge der Linie
- Selektion eines Kreisbogens ergibt den Radius des Kreisbogens
- Selektion von zwei Punkten ergibt den Punktabstand
- Selektion von Punkt - Linie ergibt ergibt den Normalabstand

- Selektion von "X(P#)" ergibt die X-Koordinate dieses Punktes (analog Y(P#) und Z(P#))
- Selektion von "X(L#)" - die X-Koordinatendifferenz dieser Linie (analog Y(P#) und Z(P#))

Die Ausgabe erfolgt als zB "VAL(12.5)"
  oder "VAL(L20)"                 # die Länge der Linie L20
  oder "VAL(Z(P20))"              # die INF_Z-Koordinate von Punkt P20



____________________________________________________________________________



Winkel (Angle)

- Eingabe eines Zahlenwertes mit der Tastatur:
  - direkt: zB "45"
  - aus Variable: zB "V20"
- Selektion eines Kreisbogens ergibt den Öffnungswinkel.
- Selektion einer Linie, eines Vektors oder von zwei Punkten
    ergibt den Winkel zur X-Achse in der XY-Ebene.


Als letzter Parameter kann "REV" angegeben werden; dadurch wird der
  Komplementärwinkel erzeugt (zB 90 wird zu 270)

Die Ausgabe erfolgt als zB "ANG(90)"
Einheit ist Grad.








Vektor / Richtung

Eingabe eines Vektors durch:

- Selektion Linie oder Polygonsegment (auch innerhalb CCV)

- Selektion von 2 Punkten

- Auswahl eines Standardvektors
  - aus den rechts unten dargestellten Vektoren
  - Eingabe mit der Tastatur (DX oder DY oder DZ oder invers dazu DIX DIY DIZ)

- Auswahl eines definierten Vektors
  - durch die PageUp/PageDown(Bild)-Tasten;
  - aus dem ObjektBrowser (links)
  - Eingabe mit der Tastatur (zB "D12")

- Eingabe eines Vektors mit der Tastatur:
  - Kartesisch (X/Y/Z-Komponenten): zB "0 0 1" (ein Z-Vektor)
  - Polar (Drehwinkel): zB "ANG(45)"
  - Polar (Drehwinkel, Kippwinkel): zB "ANG(45) ANG(30)"
    

  Eingabe eines Vektors über Winkel mit der Tastatur:
    - Drehwinkel um die Z-Achse
    ANG(120)          # X-Achse 120 Grad um Z-Achse gedreht

    - Drehwinkel (um Z-achse) und Kippwinkel (um neue Y-Achse)
    ANG(45) ANG(30)   # X-Achse 45 Grad um Z-Achse gedreht, 30 Grad gekippt


- Vektor von Winkeleingabe:


- Vektor zwischen 2 Punkten
    P20 P21      

- Vektor tangential an Kurve:
    L20         # parallel zu Linie oder Polygonsegment
    S20 0.5     # Kurve, Parameter von Punkt auf Kurve (von Auswahl Punkt auf Kurve)

- Vektor auf Ebene oder Fläche parallel zur Isoparametrischen Richtung U oder V
    R20 DRX     # X-axis von Ebene R20  (DRY, DRZ)



- Normalvektor eines Kreises oder einer Ebene oder einer Fläche

- Normalvektor auf 2 Objekte (Linie oder Punkt oder Vektor)
    DX DY    # ergibt DZ (Normalvektor)
    L20 DZ   # ergibt den Normalvektor in der Konstruktionsebene

- Normalvektor auf 3 Punkte
    P20 P21 P22       # P20 = point on X-axis, P21 = origin; P22 = near Y-axis



Als letzter Parameter kann bei Vektordefinitionen "REV" angegeben werden
  (Vektor wird verkehrt = Revers dargestellt).

Als vorletzter Parameter kann bei Vektordefinitionen
  die Vektorlänge angegeben werden.
  Beispiel: "DX 2.5" oder "DX 2.5 REV"



-------------------------------------------------------
Die Ausgabe wird eingebetted in die Typenbezeichnung "D(..)";
  z.B. "0 0 1"  wird zu "D(0 0 1)"



Beispiele:
# X-Achse 45 Grad um Z-Achse gedreht, 30 Grad gekippt, Länge 2.5
D(ANG(45) ANG(30) 2.5)










Punkt

Eingabe eines Punktes durch:
- Selektion eines Punktes
- Selektion einer Kurve - ergibt die parametrische Position am Objekt
- Selektion einer Fläche - ergibt die parametrische Position am Objekt
    (derzeit nur für B-Spline-Flächen)
- Eingabe eines Punktes mit der Tastatur:
  - Kartes. Koordinaten: "100 0 0"
  - Relativ: Basispunkt, Offset-X/Y/Z: "P22 Z(100)"
  - Relativ: Basispunkt, Richtung(Vektor), [Abstand]: "P22 D20 100"
  - Polar:  Basispunkt, Winkel, Abstand: "P22 ANG(45) 100"
  - Polar:  Basispunkt, 2 Winkel, Abstand: "P22 ANG(45) ANG(30) 100"


Die Ausgabe erfolgt als zB "P(100 0 0)"



____________________________________________________________________________



Linie

Eingabe einer Linie durch:
- Selektion einer Linie oder eines Polygonsegmentes oder
    eines linearen Segmentes in einer CCV
- Zwei Punkte definieren (anzeigen oder selektieren)
- Punkt und Vektor definieren (anzeigen oder selektieren)
- Eingabe einer Linie mit der Tastatur:
  - Horizontale Linie: Eingabe des Y-Wertes; zB "Y(10)"
  - Vertikale Linie: Eingabe des X-Wertes; zB "X(10)"
  - Zwei Punkte: zB "P(0 0 0) P(10 0 0)"
  - Punkt und Vektor: zB "P20 D(0 0 1)" oder "P20 DX"
  - Punkt und Winkel: zB "P(10 0 0) ANG(45)"
  - Punkt, Drehwinkel, Kippwinkel: zB "P22 ANG(45) ANG(30)"


Die Ausgabe erfolgt als zB "L(Y(10))"



____________________________________________________________________________



Circle

Definition eines Kreises durch:

- Selektion von Kreis oder Kreisbogen
- Selektion eines Kreissegmentes in einer CCV
- Eingabe eines Kreises mit der Tastatur:
  - Mittelpunkt, Radius: zB "P22 100"
  - Mittelpunkt, Umfangspunkt: zB "P22 P(100 0 0)"
  - Drei Umfangspunkte: zB "P21 P22 P23"
  

Die Ausgabe erfolgt als zB "C(P22 100)"
  







Ebene (Referenzsystem)

Definition einer Ebene  (position, orientation) durch:


Definition der aktiven Konstruktionsebene mit verändertem Nullpunkt:
  - Auswahl eines neuen Nullpunktes
    (es werden alle Achsen der aktiven Konstruktionsebene übernommen)
    Beispiel:
    R(P(100 0 0))


Auswahl einer Ebene, Abstand vom Nullpunkt auf Z-Achse:
  - Standardebene:
    - aus den rechts unten dargestellten Standardebenen (RZ oder RY oder RX)
    - oder Eingabe mit der Tastatur (RZ oder RY oder RX)
  - oder Auswahl definierter Ebene
  - Eingabe Abstand vom Nullpunkt auf Z-Achse]
    Beispiel:
    R(R20 50)  # wie R20 - 50 einheiten auf Z-Achse über Nullpunkt von R20



Definition einer definierten Ebene mit verändertem Nullpunkt:
  - Auswahl eines neuen Nullpunktes, Auswahl Ebene
    (es werden die Achsen dieser Ebene übernommen)
    Beispiel:
    R(R20 P22)


Definition über Nullpunkt und X-Achse:
  - Auswahl Nullpunkt
  - Auswahl Vektor für X-Achse oder Punkt auf X-Achse
    Beispiel:
    R(P22 D(L22))      # Nullpunkt P22, X-Achse from L22  
    R(P22 D(ANG(45)))  # Nullpunkt P22, aktive-X-Achse 45 Grad um Z-Achse gedreht
    # aktive-X-Achse 45 Grad um Z-Achse gedreht, 30 Grad um neue Y-Achse gekippt
    R(P22 D(ANG(45) ANG(30)))



Definition über Nullpunkt und X-Achse und Y-Achse:
  - Auswahl Nullpunkt
  - Auswahl Vektor für X-Achse oder Punkt auf X-Achse
  - Auswahl Vektor für Y-Achse oder Punkt nahe der Y-Achse
    Beispiel:
    # R(  [])
    R(P22 DZ DX)  # active Z-axis is new X-axis, active X-axis is new Y-axis 



.................................
Definition über Nullpunkt und Z-Achse (Normalvektor):
  - Eingabe "PERP"
  - Auswahl Nullpunkt
  - Auswahl Vektor für Z-Achse oder Punkt auf Z-Achse
    Beispiel: 
    # R(PERP   [] )
    R(PERP P(0 0 0) DY)


Definition über Nullpunkt und Z-Achse (Normalvektor) und X-Achse:
  - "PERP", Auswahl Nullpunkt, Auswahl Z-Achse, Auswahl X-Achse
    (Achse kann durch Vektor oder Linie oder Punkt definiert werden; siehe unten)
    Beispiel:
    R(PERP P(16 0) DIY D(DX,ANG(30))


Ebene von einem Keisbogen oder einer Ellipse:
  - Selektion von Kreis oder Kreisbogen oder eines Kreissegmentes in einer CCV
    Nullpunkt ist der Kreismittelpunkt.
  
 

---------------------------------------
Die Definition von Achsen erfordert die Eingabe von Vektoren (D..)
- siehe Definition Vektoren


Die Ausgabe von Vektoren wird eingebetted in die Typenbezeichnung "D(..)";
  z.B. D(S20 0.5)       # tangentialer Vektor an Kurve bei Parameter 0.5


Die Ausgabe der Ebene wird eingebetted in die Typenbezeichnung "R(..)";
  z.B. "P22 90"  wird zu "R(P22 90)"
