Plugin Demo_appDat1:



Plugin Demo_appDat1 shows how to store / retrieve two names.

The application-data must be written into the appdat-file.

All appdat-files ("/.appdat") will be stored in the model.
Model-load will extract this data from model into the corresponding files.

For the format in the modelfile see Format-Modelfile.

How to create / activate plugins: see Plugins.
