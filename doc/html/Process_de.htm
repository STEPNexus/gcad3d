Prozess


  Prozesse werden für NC-Bearbeitung, Simulation, Kinematik benutzt.
  Jedes Modell kann beliebig viele Prozesse haben.
  Jeder Prozess muss mit Process/Create erzeugt werden;
     dabei muss der Prozessname eingegeben werden und ein Prozessor
     ausgewählt werden. Es wird ein (leerer) Prozess erzeugt.
  Danach Aufruf von Prozessen: Process/Select-Start (oder Ctrl-P).
  Nun muss der Prozesscode erzeugt werden, siehe dazu die Dokumentation
    für den gewählten Prozessor.
  Prozesse laufen derzeit im Modus MAN.

  Prozesse können auf alle internen Funktionen zugreifen.
  Eingabefenster können systemneutral mit dem GUI-Tool "C-Kit-GUI"
    erstellt werden.
  Prozessoren sind Lademodule, kompiliert (Programmsprache C oder andere).
  Jeder Prozessor kann eine spezifische Befehlstabelle haben.
  Die Prozessdaten werden im Modell in "SECTION PROCESS {Prozessname}"
    gespeichert.
