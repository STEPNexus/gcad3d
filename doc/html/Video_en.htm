How to create gcad3d-tutorial-videos


____________________________________________________________________________
Prerequisites:
- install simplescreenrecorder


Start gcad3d
- set ScreenRes -> 1920 X 1020
- gcad-window to left, full height, 70 % width;
- Options ScreenCast ON

Start simplescreenrecorder
  - select the window you want to record
  - audio off;
  --
  - store as xx.mp4
  - Preset medium
  --
  - start recording in icon in toolbar on top
  ..
  - stop Crecording




____________________________________________________________________________
OLD_VERSION:

Prerequisites:
- install gtk-recordmydesktop

Start gcad
  - activate Options / ScreenCast ON

Start gtk-recordmydesktop
  - audio off;
  - select the window you want to record
  - start recording
  - stop (select the icon (red square) on top
  - wait for the output file out.ogv, save.
