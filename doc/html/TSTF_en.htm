Testtool TSTF



- can be used by plugin
- can process codes and commands read in from a file
- can execute remote-commands (eg load model ..)
- can add and process cad-codes
- can stop / continue program (WAIT_ESC)
- can dump data (DUMP ())
- can call system-programs (edit (input-)file ..)
- provides callback for private testfunctions (TSTF_cb())
- can process subroutines


Test_it:
- set Option/compile DLL ON   (or cadstart with parameter "comp")
- select Plugin / Demo_tstf_1
- modify inputfile (eg ../APP/Demo_tstf_1.dat) without stopping gcad;
  - set appropriate filename in ../APP/Demo_tstf_1.c
- Ctrl-P will rebuild and restart the plugin 


See demoprogram (plugin Demo_tstf_1 - ../APP/Demo_tstf_1.c).
